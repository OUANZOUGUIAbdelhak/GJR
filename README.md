## Deploy to GitHub Pages

1. **Create a new repository** on GitHub (e.g., `valentine` or `jasmine-valentine`)

2. **Initialize git and push** (run in this folder):

   ```bash
   git init
   git add .
   git commit -m "Initial commit - Valentine page"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git push -u origin main
   ```

3. **Enable GitHub Pages**:
   - Go to your repo on GitHub
   - Click **Settings** â†’ **Pages** (in the left sidebar)
   - Under "Source", select **Deploy from a branch**
   - Branch: **main**, Folder: **/ (root)**
   - Click **Save**

4. **Your page will be live at**: `https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/`
